## Featured Projects

### Metaphor Detection Using Multi-Classifier Bagging

Built an **NLP-based machine learning model** to classify text as metaphorical or literal across multiple metaphor categories. The project uses TF-IDF vectorization and a custom ensemble learning approach to improve classification performance.

**Tech Stack:** Python, Pandas, NumPy, Scikit-learn, TF-IDF, NLP, Random Forest, Gradient Boosting, Naive Bayes, Logistic Regression

**Highlights:**

- Developed a text classification model for metaphor detection using NLP techniques
- Applied **TF-IDF vectorization** with unigram and bigram features
- Built a custom **Multi-Classifier Bagging Ensemble**
- Combined Random Forest, Gradient Boosting, Naive Bayes, and Logistic Regression
- Used majority voting and probability averaging for final predictions
- Achieved **86% accuracy**, **0.856 macro precision**, and **0.77 macro F1 score**
